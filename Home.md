## 🏗️ Platform Overview

NEXUS HRMS is an enterprise-grade Human Resource Management System built with modern web technologies:

| Component | Technology |
|-----------|-----------|
| Frontend | Next.js 16, React 19, TypeScript 5, Tailwind CSS 4 |
| UI Library | shadcn/ui (Radix UI primitives) |
| Backend | Next.js API Routes |
| Database | PostgreSQL (Neon Serverless) + Prisma ORM |
| State Management | Zustand |
| Deployment | Vercel with GitHub CI/CD |

## 📦 Modules (25+)

The platform includes 25+ integrated modules covering the complete HR lifecycle:

- **Overview**: Dashboard & Analytics
- **People**: Employees, Onboarding
- **Recruitment**: ATS, Job Portal, AI Interview
- **Time**: Attendance, Leave Management
- **Compensation**: Payroll Processing
- **Growth**: Performance, Learning & Development
- **Engagement**: Employee Engagement, Helpdesk
- **Operations**: Travel & Expense, Assets, Compliance
- **Automation**: Workflow Builder
- **Ecosystem**: Client Portal, Vendor Portal, Sub-Vendor Portal
- **AI**: AI Chatbot, AI Interview
- **Admin**: Companies, Audit Logs, Settings
- **Help**: Help & Training Center
