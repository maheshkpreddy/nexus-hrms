# Module Guide

## Dashboard
The main overview screen showing key HR metrics, department distribution, recent activities, and quick action shortcuts.

**Key Features:**
- Real-time employee count and distribution
- Department-wise headcount analytics
- Attrition rate tracking
- New hire pipeline
- Pending approvals count

## Employees
Complete employee lifecycle management from onboarding to offboarding.

**Key Features:**
- Grid and list view options
- Department and status filters
- Quick search across all fields
- Employment type tracking (Full-time, Part-time, Contract, Intern)
- Add/Edit employee dialogs with department and branch assignment

**Workflow:**
1. Navigate to Employees
2. Click "Add Employee" to create new record
3. Fill in personal and professional details
4. Assign department and branch
5. Set employment type and status
6. Save the record

## Recruitment (ATS)
Full applicant tracking system with Kanban-style pipeline management.

**Key Features:**
- Job posting creation and management
- Candidate pipeline tracking
- Interview scheduling
- Offer management
- Pipeline analytics

## Job Portal
External-facing job portal for candidates to browse and apply for open positions.

## AI Interview
AI-powered interview system with automated question generation and candidate scoring.

## Attendance
Daily attendance tracking with check-in/check-out functionality.

**Key Features:**
- Web-based check-in/check-out
- Attendance reports and analytics
- Shift management
- Overtime calculation

## Leave Management
Comprehensive leave management with multi-type support.

**Key Features:**
- Multiple leave types (Casual, Sick, Earned, etc.)
- Automatic balance calculation
- Manager approval workflow
- Leave calendar

## Payroll
Monthly payroll processing with tax computation and statutory compliance.

**Key Features:**
- Automated salary calculation
- Tax computation and deductions
- Payslip generation
- Bank file generation
- Statutory compliance reports

## Performance

- Appraisal management
